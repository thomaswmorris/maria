    "# Polarized observations\n",
    "\n",
    "This tutorial covers working with polarized instrument and maps, and recovering polarized maps from observations."

  {
   "cell_type": "markdown",

   "source": [
    "We start with a normal instrument, and create two orthogonally polarized copies of each detector by setting ``polarized: True`` in the ``Array`` config:"
   ]
  },
  {
   "cell_type": "code",
   "execution_count": null,
   "id": "2",
   "metadata": {},
   "outputs": [],
   "source": [
    "import maria\n",
    "from maria.instrument import Band\n",
    "\n",
    "f150 = Band(\n",
    "    center=150e9, \n",
    "    width=30e9, \n",
    "    NET_RJ=60e-6, \n",
    "    knee=1e0, \n",
    "    gain_error=5e-2)\n",
    "\n",
    "array = {\"field_of_view\": 0.2, \n",
    "         \"shape\": \"circle\", \n",
    "         \"beam_spacing\": 1.8,\n",
    "         \"primary_size\": 10, \n",
    "         \"polarized\": True,\n",
    "         \"packing\": \"sunflower\",\n",
    "         \"bands\": [f150]}\n",
    "\n",
    "instrument = maria.get_instrument(array=array)\n",
    "\n",
    "print(instrument.arrays)"
   ]
  },
  {
   "cell_type": "markdown",
   "id": "3",
   "metadata": {},
   "source": [
    "We can see the resulting polarization footprint in the instrument plot:"
   ]

  {
   "cell_type": "code",
   "execution_count": null,
   "id": "4",

   "outputs": [],
   "source": [
    "print(instrument)\n",
    "instrument.plot()"
   ]
  },
  {
   "cell_type": "markdown",
   "id": "5",
   "metadata": {},
   "source": [
    "Let's observe the use the Einstein map, which has a faint polarization signature underneath the unpolarized signal of Einstein's face. Remember that all maps are five dimensional (stokes, frequency, time, y, x); this map has four channels in the stokes dimensions (the I, Q, U, and V [Stokes parameters](https://en.wikipedia.org/wiki/Stokes_parameters)). We can plot all the channels by giving ``plot`` a shaped set of stokes parameters."
   ]
  },
  {
   "cell_type": "code",
   "execution_count": null,
   "id": "6",
   "metadata": {},
   "outputs": [],
   "source": [
    "input_map = maria.map.get(\"maps/einstein.h5\")\n",
    "input_map.data *= 4\n",
    "print(input_map)\n",
    "\n",
    "\n",
    "input_map.plot(slices={\"stokes\": [[\"I\", \"Q\"], \n",
    "                                  [\"U\", \"V\"]]})"
   ]
  },
  {
   "cell_type": "code",
   "execution_count": null,
   "id": "7",
   "metadata": {},
   "outputs": [],
   "source": [
    "from maria import Planner\n",
    "\n",
    "planner = Planner(target=input_map, site=\"mauna_kea\", constraints={\"el\": (45, 90)})\n",
    "\n",
    "plans = planner.generate_plans(\n",
    "                               total_duration=1800,\n",
    "                               max_chunk_duration=1800,\n",
    "                               sample_rate=50, \n",
    "                               scan_pattern=\"raster\",\n",
    "                               scan_options={\n",
    "                                   \"n\": [(15, 1), (1, 16)],\n",
    "                                   \"radius\": 0.5 * input_map.width.deg,\n",
    "                                   \"speed\": 0.5,\n",
    "                               })\n",
    "\n",
    "plans[0].plot()\n",
    "print(plans)"
   ]
  },
  {
   "cell_type": "code",
   "execution_count": null,
   "id": "8",
   "metadata": {},
   "outputs": [],
   "source": [
    "sim = maria.Simulation(\n",
    "    instrument,\n",
    "    plans=plans,\n",
    "    site=\"mauna_kea\",\n",
    "    atmosphere=\"2d\",\n",
    "    map=input_map,\n",
    ")\n",
    "\n",
    "print(sim)"
   ]
  },
  {
   "cell_type": "code",
   "execution_count": null,
   "id": "9",
   "metadata": {},
   "outputs": [],
   "source": [
    "tods = sim.run()\n",
    "\n",
    "print(tods)\n",
    "tods[0].plot()"
   ]
  },
  {

    "    },\n",
    "    tods=tods,\n",
    "    bilinear=False,\n",
    ")\n",
    "\n",
    "mapper.map.plot(slices=\"all\")"
